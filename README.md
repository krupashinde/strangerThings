# Stranger Things – Interactive Web Experience

A visually immersive **Stranger Things–inspired interactive landing page** developed using **React and Vite**, designed to showcase modern frontend development practices, smooth animations, and engaging user interactions.

This project demonstrates a strong focus on **UI/UX design**, **component-based architecture**, and **animation-driven storytelling**.

---

## Overview

The goal of this project is to recreate a cinematic web experience inspired by the *Stranger Things* universe. It leverages modern frontend technologies to deliver smooth transitions, interactive hover effects, and a dark-themed visual aesthetic.

The application is built with scalability and performance in mind using Vite’s fast development environment.

---

## Technology Stack

- **Frontend Framework:** React (Vite)
- **Styling:** CSS3
- **Animations:** Framer Motion
- **Language:** JavaScript (ES6+)
- **Build Tool:** Vite
- **Version Control:** Git & GitHub

---

## Key Features

- Interactive hero section with mouse-tracking reveal effects  
- Smooth component animations powered by Framer Motion  
- Dark, cinematic UI inspired by Stranger Things  
- Modular and reusable component structure  
- Responsive layout across devices
